# Sovereign Rollup CDK Starter

This is an empty AWS CDK project set up with TypeScript.

## Getting Started

- Run `npm run build` to compile TypeScript
- Run `npx cdk deploy` to deploy your stack
- Add AWS resources to `lib/sov-rollup-cdk-starter-stack.ts`

## Useful Commands

- `npm run build` - Compile TypeScript to JavaScript
- `npm run watch` - Watch for changes and compile
- `npm run cdk` - Run CDK commands
- `npx cdk synth` - Synthesize CloudFormation template
- `npx cdk diff` - Compare deployed stack with current state
- `npx cdk deploy` - Deploy this stack to your AWS account/region
- `npx cdk destroy` - Destroy the stack
